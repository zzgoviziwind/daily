## 📝 使用建议

### 任务进度追踪
- 按月创建日志文件，格式：`YYYY-MM-任务日志.md`
- 每日简单记录完成的任务
- 月度结束后归档到 `50-Archives/`

- 记录实际使用过的命令
- 添加使用场景说明
- 定期整理删除不再使用的
