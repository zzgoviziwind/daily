# Claude Code Memory

自动记忆的系统配置文件。

## 任务追踪配置

| 配置项 | 值 |
|--------|-----|
| 任务目录 | `30-Areas/ClaudeCode-任务追踪/` |
| 模板文件 | `Templates/Task-任务模板.md` |
| 保留策略 | 进行中任务永久保留，已完成任务 30 天后归档 |
| 命名规范 | `TASK-YYYYMMDD-序号.md` |

## 会话记录

最近会话会自动记录到对应的任务笔记中。

## 使用说明

1. 开始新任务时，创建任务笔记
2. 每次会话结束时，更新任务笔记
3. 任务完成后，移动到已完成文件夹
4. 定期清理已过期的任务
